# üç™ Cookie Sorting Game

A fun and interactive browser game designed to teach kids the concept of **variables** through hands-on play. Players sort cookies into jars based on their flavors, learning the fundamentals of randomization and assignment in a playful context.

This project was created as part of the **CLC Literary Randomizer Build (Spike Essential)** to introduce children to basic programming concepts.

---

## üéÆ Demo

Currently runs locally. Open `index.html` in a web browser to play.

---

## üß© Features

- **Hover Tooltips:** See the flavor of each cookie by hovering over it.  
- **Drag & Drop:** Drag cookies into the correct jar.  
- **Audio Feedback:**  
  - Correct placement: plays a success sound  
  - Wrong placement: plays an error sound  
  - Completing all cookies: celebratory sound and confetti effect  
- **Dynamic Resizing:** Cookies in jars resize automatically to fit neatly.  
- **Responsive Layout:** Works on various screen sizes for flexible use.
